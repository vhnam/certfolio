## Build with AI: Knowledge Hub

### Lab Overview

- Purpose: Use **NotebookLM** to create a centralized knowledge hub for project documents.
- Requirement: Sign into NotebookLM with a Google Account.
- Approach: Upload project sources and ask questions using natural language.

NotebookLM analyzes the uploaded materials and provides answers grounded in the original documents.

### Step 1: Ask Questions About Project Documents

Create a new notebook and upload project materials.

Process:

1. Open NotebookLM and select **Create a new notebook**.
2. Upload project-related documents.

Possible document sources:

- Project memo
- Work-back plan
- Packaging plan
- Creative brief

Additional options:

- Upload files from Google Drive.
- Search the web for relevant sources.
- Add more documents anytime from the left sidebar.

Once the documents are uploaded, NotebookLM generates a workspace where you can query the information.

1st Example query:

```
When are we starting [key phase in your project, e.g., full-scale packaging production]?
```

2nd Example query:

```
What are the key milestones of this project?
```

#### Asking Questions

Use the input field to ask questions about the project.

Example queries:

- When are we starting full-scale packaging production?
- What are the key milestones of this project?

NotebookLM responses include:

- Extracted answers from the uploaded documents
- Source citations linking back to the original materials

This ensures answers remain **accurate and traceable to the source documents**.

### Step 2: Share the Notebook with Your Team

Allow team members to access and query the knowledge hub.

Process:

1. Click the **Share** button in the notebook.
2. Enter team members’ email addresses.
3. Set access permissions.
4. Optionally add a **Welcome Note**.
5. Keep **Notify people** enabled and click **Save**.

Team members will receive an email with a link to access the notebook.

### Applying the Workflow

NotebookLM helps teams manage complex projects by:

- Centralizing project documents
- Allowing natural language queries
- Providing answers with document citations
- Enabling shared access for team collaboration
